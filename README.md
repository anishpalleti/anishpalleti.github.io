# New Website (HTML starter)

Simple static starter site with HTML, CSS, and JavaScript.

Preview locally:

```bash
# open directly (double-click) or run a simple HTTP server
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Files:

- [index.html](index.html) — main page
- [styles.css](styles.css) — styles
- [script.js](script.js) — small interactions

Feel free to edit the files to customize content and styling.
